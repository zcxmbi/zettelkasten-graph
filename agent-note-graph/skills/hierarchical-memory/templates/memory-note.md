# 记忆笔记模板

在知识体系文件夹（`<vault-path>`）中创建。替换 `[]` 占位符。

```markdown
---
tags: [领域标签]
source: [来源URL或书籍，无则删除此行]
created: YYYY-MM-DD
links_to: []
linked_from: []
---

# [笔记标题]

> [一句话核心观点]

## 定义

[概念的精确定义，1-2句话]

## 核心内容

[正文部分]

## 链接

- [[相关笔记A]] — [关联说明]
- [[相关笔记B]] — [关联说明]
```

## 写作约定

- **原子性**：一条笔记只记一个概念
- **自包含**：脱离上下文也能看懂
- **链接优先**：用 `[[链接]]` 代替重复解释
- **tags 用领域标签**（如 `游戏设计`、`工具`），不写 `参考记忆`/`活跃记忆`——层级是 Hermes 体系内部的事
- **`usage_count` 和 `tokens_at_last_use`** 由执事在 Hermes 体系内追踪，不在知识体系笔记中维护
