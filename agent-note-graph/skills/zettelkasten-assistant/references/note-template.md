# 原子笔记模板

复制此模板创建新笔记。替换 `[]` 占位符为实际内容。

```
---
tags: [#领域 #子标签]
source: [来源URL或书籍页码，无来源则删除此行]
---

# [笔记标题]

> [一句话核心观点]

## 定义

[概念的精确定义，1-2句话]

## 核心原理

- [原理1]
- [原理2]

## 关键洞察

- [洞察1]
- [洞察2]

## 适用场景

- [场景1]
- [场景2]

## 相关概念

- [[相关笔记A]] — [一句话关联说明]
- [[相关笔记B]] — [一句话关联说明]
```

## 写作约定

- **原子性**：一条笔记只讲一个概念。如果发现笔记在讲两件事，拆成两条。
- **自包含**：即使脱离上下文，读者也能看懂这条笔记在说什么。
- **链接优先**：能用 `[[链接]]` 的地方不重复解释——让网络本身承载知识。
- **标签在 YAML 头**：统一用 `tags:` 字段，不用正文第一行。Zettlr 侧边栏「相关文件」能识别。
- **source 可选**：有外部来源时记录，自己总结的删掉该行。
- **一句话摘要**：`>` 引用块写核心观点，方便日后扫读。
